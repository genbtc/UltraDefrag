Compilation

UltraDefrag uses Windows SDK v7.1 to build binaries for all the supported processor architectures. To build 32-bit binaries MinGW can also be used.

In addition to the compilers a few other packages need to be installed as well:

  • wxWidgets 2.8.12 to build GUI upon it
  • NSIS compiler to create the installers
  • Lua to run build scripts
  • 7-zip file archiver to compress the source code and portable packages
  • Doxygen to create the documentation
  • Basic MiKTeX Installer to create the optional PDF documentation
  • md5sum tool to create the checksums
  • UPX to reduce the size of the installers
  • Transifex Client to download the updated translations from the translation project
  • GNU gettext to extract and compile the translations

So,

  • Install wxWidgets, NSIS, 7-zip, Doxygen and optionally MiKTeX and "GNU gettext".
  • Extract Lua to some directory on your %PATH%, e.g. to C:\Windows.
  • Place md5sum.exe into the src\tools folder of the source code distribution.
  • Extract upx.exe into the src\tools\upx folder of the source code distribution.
  • Place tx.exe into the src\tools\transifex folder of the source code distribution.

If you need an IDE for development install Code::Blocks as well.

To properly set up the build environment edit the setvars.cmd script and then type:

patch-tools

To update the translations type:

update-translations

To build wxWidgets library type:

wxbuild

Compile it by all the compilers you use for UltraDefrag compilation to be able to link binaries correctly. Also don't forget to recompile the library again after any adjustments of src\tools\patch\wx\setup.h file. This file is included exactly to build wxWidgets automatically with options suitable for UltraDefrag.

To build UltraDefrag using defaults type:

build

To list all the supported options type:

build --help

To select one of the predefined build actions type:

build-menu

To start Code::Blocks IDE type:

start-CodeBlocks

If you'd like to know, how the build process works, look at .cmd files - they contain the complete information.
