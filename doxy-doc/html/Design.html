Design Notes

We have used the following excellent guides of software design:

  • The Art of Unix Programming
  • GNU coding standards
  • Information for maintainers of GNU software

The volume processing algorithms are contained in udefrag.dll library.

Both command line and graphical interfaces are built on top of wxWidgets library which helps a lot to do all the things in the most simple manner.

The boot time interface is written in pure C simply because it's intended to be executed in an environment which misses some stuff needed for more sophisticated things. However, the interface is built over ZenWINX library which makes its development easier.

And, at last, Lua is used to convert reports from an internal representation to HTML pages and plain text files.
